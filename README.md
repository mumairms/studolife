# 🚀 StudoLife - Productivity App

A full-stack productivity web application to manage tasks, track progress, and stay organized.

---

## 🌐 Live Demo

🔗 Frontend: https://studolife.netlify.app  
🔗 Backend: https://productivity-backend-ccxh.onrender.com  

---

## 🛠 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Recharts
- Framer Motion

### Backend
- Node.js
- Express.js
- MongoDB (Atlas)
- JWT Authentication

---

## ✨ Features

- 🔐 User Authentication (Login/Register)
- 📋 Task Management (Add / Delete / Complete)
- 📊 Dashboard Analytics
- 🌙 Dark / Light Mode
- ⚡ Real-time UI Updates
- ☁️ Deployed on Netlify & Render

### 1️⃣ Clone Repository

```bash
git clone https://github.com/mumairms/studolife.git
cd studolife


Backend Setup:
cd backend
npm install

Create .env:
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret

Run:
node server.js

Project Structure:
Frontend Setup:
cd productivity-app
